1.Gitのどの機能が最も便利だと思いましたか？
git reset --soft HEAD^　でコミットを取り消しできるのが便利でした


2.コンフリクトを解決するときに困ったことは？
どのブランチのどのページでなんの変更が問題になっているのかを整理するのが難しいと感じました


3.mergeとrebaseの違いを自分の言葉で説明してください
merge：枝分かれして進んだ記録が残る
  　　　mainとbranchが別々に進んで、合流した時にどこで合流したかの履歴が残る
rebase：一本道に並べ替える
      　branchをmainに付け直すことで枝分かれしていない一本線のような形になる


4.明日から実際の開発で使いたいGitコマンドは？
git branch　でどこのブランチの作業をしているのか認識するために頻繁に使用していたので、明日以降も
混乱しそうな時は使っていきたいです。
